## position 的属性和原理
  和文档流有什么关系?
  - relative 相对定位
    - 文档流中相对默认的位置 移动 left top...
    - 盒子原来的占位（文档流中的位置和大小）不受影响
  - absolute 绝对定位
    - 绝对定位会离开文档流
    - 找到最近的 祖先有定位节点的元素
      如果没找到的化 相对于body定位
